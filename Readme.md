# 🏆 DecisionX – FIFA World Cup Optimizer

DecisionX is an end-to-end sports analytics and decision-support platform designed for FIFA World Cup tournament analysis. The project integrates Machine Learning, Monte Carlo Simulation, and Operations Research techniques to predict match outcomes, simulate thousands of tournament scenarios, and optimize squad selection strategies.

Unlike traditional football prediction systems that only forecast match results, DecisionX focuses on decision optimization. Match outcome probabilities generated by predictive models are used as inputs for large-scale tournament simulations and optimization models, enabling strategic analysis of team performance and championship likelihood.

## ✨ Key Features

- Match Outcome Prediction using Machine Learning
- FIFA World Cup Tournament Simulation (10,000+ Monte Carlo Runs)
- Championship Probability Estimation
- Team Performance Analytics
- Squad Selection Optimization using Integer Programming
- Interactive Streamlit Dashboard
- Data-Driven Scenario Analysis

## 🛠 Technologies Used

- Python
- Pandas & NumPy
- Scikit-Learn
- XGBoost
- Plotly
- Streamlit
- PuLP / OR-Tools
- Monte Carlo Simulation

## 🎯 Project Objective

To develop a decision-support system that helps evaluate tournament strategies, quantify championship probabilities, and optimize team selection through the integration of predictive analytics and Operations Research methodologies.

## 📈 Results

- Predicted international football match outcomes using machine learning models.
- Simulated 10,000+ FIFA World Cup tournaments to estimate stage-wise qualification probabilities.
- Generated data-driven squad recommendations using optimization techniques.
- Built an interactive analytics dashboard for tournament forecasting and strategic decision-making.
