DecisionX is an end-to-end sports analytics and decision-support platform designed for FIFA World Cup tournament analysis. The project combines **Machine Learning**, **Monte Carlo Simulation**, and **Operations Research** techniques to predict football match outcomes, simulate thousands of tournament scenarios, and optimize squad selection strategies.

Unlike traditional football prediction systems that only forecast match results, DecisionX uses predictive models as inputs for tournament simulations and optimization models, enabling strategic decision-making.

**The project allows users to:**

- Predict football match outcomes using Machine Learning
- Simulate complete FIFA World Cup tournaments (10,000+ runs)
- Estimate championship and qualification probabilities
- Optimize squad selection using Integer Programming
- Analyze tournament scenarios through an interactive dashboard

## Interactive Web Application

The project includes an interactive **Streamlit dashboard** that allows users to explore tournament predictions and optimization results without running the notebook manually.

The dashboard enables users to:

- Predict match outcomes between international teams
- Run Monte Carlo simulations of FIFA World Cup tournaments
- Visualize championship and qualification probabilities
- Explore team performance analytics
- Generate optimized squad recommendations
- Compare different tournament scenarios interactively

The interactive dashboard provides an intuitive way to analyze predictive models and optimization results through dynamic visualizations.
